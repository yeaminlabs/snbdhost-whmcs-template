# Changelog

All notable changes to the SNBDHost WHMCS theme will be documented in this file.

## [1.0.3] - 2026-06-20
### Fixed
- Added a top important notification section to the WHMCS Admin Dashboard.
- Dashboard now displays the last reported bug and what bug was fixed in the latest version.

## [1.0.2] - 2026-06-20
### Fixed
- Split update buttons in the manager module to allow updating theme and module separately.
- Overhauled the manager module UI to use SNBDHost branding colors.

## [1.0.1] - 2026-06-20
### Fixed
- Fixed template path resolution error in updater module.
- Added capability for the updater script to automatically update the addon module alongside the theme.

## [1.0.0] - 2026-06-15
### Added
- Initial release of the SNBDHost Portal Template.
- Custom login and checkout pages.
- Client area enhancements.
